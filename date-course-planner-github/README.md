# 데이트 플래너

GitHub Pages에 올릴 수 있는 Vite + React 앱입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 배포 방법

1. GitHub에서 새 저장소를 만듭니다.
2. 이 ZIP 안의 파일들을 저장소에 업로드합니다.
3. GitHub 저장소에서 Settings → Pages로 들어갑니다.
4. GitHub Actions 방식으로 배포하거나, 로컬에서 `npm run build` 후 `dist` 폴더를 배포합니다.

## 아이콘

흰 배경에 빨간 하트 아이콘이 포함되어 있습니다.
